Multiple Authentication in Laravel 5.6

Step: 1 
--------

Download from GitHub:
https://github.com/amanat-juwel/multiauth56.git

--------
Step: 2 
--------
run command: composer update
rename .env.example with .env
configure database on .env file
create a new database with name multiauth_56
create new tables with: php artisan migrate
php artisan key:generate
